# Preferencias de documentación

Toda la documentación del proyecto se encuentra escrita en formato Markdown compatible con Visual Studio Code.

Los archivos principales incluyen:

- README.md
- DATABASE.md
- WORKFLOWS.md

La documentación describe:

- Objetivos del proyecto.
- Metodología estadística.
- Modelos implementados.
- Resultados obtenidos.
- Procedimientos de ejecución.
- Estructura del proyecto.

Además, se documentan técnicas avanzadas de Ciencia de Datos tales como:

- PCA (Principal Component Analysis).
- K-Means Clustering.
- Método del Codo.
- Coeficiente Silhouette.
- UMAP (Uniform Manifold Approximation and Projection).
- Regresión Lineal.
- ANOVA.
- Kruskal-Wallis.
- Tamaño del efecto (η²H).
